# Local Data Workspace

This folder is for local crater CSV exports and generated analysis outputs.

- Put exported or imported crater review CSV files in `data/csv/`.
- Put generated analysis folders in `data/analysis/`. The GUI does this automatically when analyzing CSV files from `data/csv/`.
- CSV files, videos, and analysis outputs are ignored by git.

The application can still open or export files anywhere. This folder is only a convention to keep the repository root tidy.
